
# 《北极星 V2 Epic / Story / Task 结构》v0.1

---

## 1. Epic 层（大功能块）

| Epic ID | 名称                   | 描述                                                        |
| ------- | ---------------------- | ----------------------------------------------------------- |
| E1      | Core 基础服务升级      | 在 V1 基础上增加项目、环境、集合、运行记录服务              |
| E2      | Web Console 扩展       | V2 版本页面升级，支持项目页、集合页、运行记录页、环境管理页 |
| E3      | Browser Extension 扩展 | 最近项目快捷入口，显示当前活跃项目，当前站点快速加入规则    |
| E4      | MCP Server 项目级能力  | 提供项目级工具和资源，支持查询项目、环境、集合、运行记录    |

## 4. 使用建议

1. **Epic -> Story -> Task 层层拆分**，每个 Task 可以直接交给 vibe coding 生成代码。
2. **Task 层可以包含**：

   * 数据类型定义（shared-types）
   * Core 接口实现
   * Storage Adapter 持久化
   * Console / Extension 页面
   * MCP 工具实现
   * 扩展 hook / UI slot 占位
3. **保持版本边界**：V2 不包含 V3 功能（历史版本、对比、问题现场、分享）。
4. **可迭代生成**：先生成 Core，再生成 Console，再生成 Extension，最后生成 MCP。

你是一名全栈开发助理，任务是生成北极星（Polaris）项目 V3 的代码。

【项目背景】
北极星 V3 是在 V1 / V2 基础上升级的本地接口资产平台。  
主要目标：
- 支持历史版本管理（请求、Mock、集合）
- 支持版本对比
- 支持运行结果对比
- 支持问题现场（IssueBundle）创建和管理
- 支持轻量只读分享（请求 / Mock / 运行记录 / 问题现场）
- 支持项目摘要展示（首页、项目详情页）
- MCP 提供完整项目上下文（历史版本、运行记录、问题现场、项目摘要）
- Web Console、Browser Extension、Local Core 目录结构沿用 V1 / V2
- UI 保持通俗、轻量、易上手
- 不内置 AI，不开放第三方插件市场

V3 不包含：
- 内置 AI
- 第三方插件市场
- 企业权限体系
- 高门槛脚本系统
- 复杂自动化测试平台

---

【技术边界与架构】
沿用 V1 / V2 monorepo 结构：

polaris/
  apps/
    core/        # Local Core
    console/     # Web Console
    extension/   # 浏览器插件
  packages/
    shared-types/
    shared-contracts/
    mcp-contracts/
    shared-utils/
    extension-sdk/
  docs/
  scripts/
  configs/

模块职责：

1. Core：
   - Version Service（资产历史版本记录与查询）
   - Diff Service（请求版本 / Mock / 集合 / 运行结果对比）
   - Issue Bundle Service（问题现场创建、查询、复制、删除）
   - Share Service（轻量只读分享）
   - Summary Service（项目摘要）
   - MCP Server（V3 项目级 tools / resources）
   - Storage Adapter
   - 扩展宿主预留

2. Console：
   - 首页（最近项目、最近失败运行、最近问题现场、最近变更摘要）
   - 项目详情页（项目摘要、最近请求、最近运行、环境、规则）
   - 请求资产页（版本管理、问题现场、分享入口）
   - Mock 页（历史版本、分享入口）
   - 集合详情页（运行、版本、问题现场、分享入口）
   - 运行记录页（列表 + 详情 + 对比）
   - 问题现场页、问题现场详情页
   - 设置页（扩展能力说明、MCP 高级能力说明）

3. Extension：
   - 当前活跃项目展示
   - 当前站点加入当前项目规则
   - 最近项目快捷入口
   - 最近问题现场快捷入口
   - 保持轻量，不做编辑能力

4. Shared-types / contracts：
   - AssetVersion、IssueBundle、ShareRecord、ProjectSummary
   - 枚举、常量、状态
   - Core / Console / Extension 统一使用
   - MCP V3 tools / resources schema
